# Data Analytics Portfolio

I’m Tarek, a final-year Computer Science student in the UK targetting junior/graduate Data Analyst / BI Analyst roles.

## What’s inside
- **01_sql/** — SQL practice, case studies, and interview-style questions.
- **02_excel/** — Excel analysis packs (cleaning, pivots, formulas, dashboards).
- **03_powerbi/** — Power BI reports (PBIX), model notes, and insights.
- **assets/screenshots/** — Report screenshots used in READMEs.

## How to open
- **SQL:** open `.sql` files in VS Code (or any editor).
- **Excel:** open `.xlsx` in Microsoft Excel.
- **Power BI:** open `.pbix` using **Power BI Desktop (Windows)**.
